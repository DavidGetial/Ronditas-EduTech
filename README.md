# Ronditas EduTech

Plataforma educativa para grados cuarto y quinto.  
Permite a los alumnos subir talleres en cualquier formato (Word, PDF, imágenes) y responder cuestionarios en línea.  
Los maestros pueden visualizar archivos sin descargarlos, calificarlos y dar retroalimentación.  
Incluye panel de control para ver entregas y progreso.

## Tecnologías
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Base de datos: Supabase (PostgreSQL)
- Control de versiones: GitHub

## Instalación
1. Clonar el repositorio
2. Instalar dependencias con `npm install`
3. Configurar variables en `.env` (SUPABASE_URL, SUPABASE_KEY)
4. Ejecutar con `npm run dev`
